# Evoluted-PHP-test
Contains the source code in laravel for the php engineer technical test of evoluted.

## Requirements

- PHP 8.4.10
- Composer

## Installation

Install dependencies by running in root folder on CLI:

composer install

## Script 1 – Parse Input to JSON

Run in root folder on CLI:

php artisan cards:parse

## Script 2 – Calculate Points

Run in root folder on CLI:

php artisan cards:points

## Running Tests

php vendor\phpunit\phpunit\phpunit
